Tags que usei - função

  • <a> - Cria um link para outra página.
  • <br> - Insere uma quebra de linha.
  • <body> - Contém o conteúdo visível da página.
  • <div> - Define uma seção ou container de conteúdo.
  • <head> - Contém metadados sobre o documento.
  • <header> - Representa o cabeçalho da página.
  • <h1> - Define o título principal da página.
  • <h2> - Define um título de nível 2, normalmente usado para seções.
  • <h3> - Define um subtítulo de nível 3.
  • <html> - Elemento raiz que contém todo o conteúdo HTML.
  • <img> - Exibe uma imagem.
  • <li> - Define um item em uma lista, como esta que está sendo exibida.
  • <link> - Importa arquivos externos, como folhas de estilo CSS.
  • <meta> - Define informações sobre o documento, como o conjunto de caracteres.
  • <p> - Define um parágrafo de texto.
  • <title> - Define o título da página exibido na aba do navegador.
  • <!DOCTYPE html> - Define o documento como HTML5.
